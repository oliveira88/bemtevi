> Bem-te-vi!

## Compatibility

- Linux
- Windows
- Mac

## Installing

Install globally:

    npm install -g bemtevi

## Running

Run command:

    bemtevi

Or import in NodeJS script:

    import bemtevi from 'bemtevi';

    bemtevi();

## Running with [npx](https://www.npmjs.com/package/npx)

Run command:

    npx bemtevi
